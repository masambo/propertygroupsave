# Property Group Save Namibia

Static website for Property Group Save Namibia.

## Hosting

This site can be hosted as a static site. The deploy root is the repository root.

- Entry file: `index.html`
- Styles: `styles.css`
- Scripts: `script.js`
- Assets and application form: `public/`

## GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open **Settings > Pages**.
3. Select the `main` branch and `/ (root)` as the source.
4. Save and wait for GitHub Pages to publish.

## Netlify or Vercel

Use the included static build configuration:

- Build command: `npm run build`
- Output directory: `dist`
